What is this?

A UI style guide, pattern library, and theme based on Bootstrap and Material Design. Our docs are packed full of usability guidelines, examples, and source code. We owe a great deal to the open source work of  USPTO's Design Library

How do I get it?

Please see our theme on GitHub for instructions on how to use tht theme.  GitHub Repository

Who is it for?

This is intended for designers and developers working on CBP's Office of Information and Technology (OIT) related projects. But, anyone can take advantage of it. If it's useful to you, use it.

Why did we make it?

We have a lot of web projects at the CBP. These design guidelines and resources aim to create a unified experience across all projects at the CBP, both internal and external.
